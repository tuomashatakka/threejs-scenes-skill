FrameContext interface

Per-frame context handed to every animated subsystem by the frame loop. `delta` is seconds since the last frame, `elapsed` is total seconds, `frame` is a monotonically increasing frame counter.

interface FrameContext {
    delta: number;
    elapsed: number;
    frame: number;
}

FrameLoop interface

Self-contained Clock-driven frame loop. Every animated subsystem registers through `onFrame` and receives a single shared {@link FrameContext}. Mirrors the inlined loop in the templates — no exotic dependency.

interface FrameLoop extends Disposable {
    onFrame(cb: FrameCallback): () => void;
    registerUpdate(cb: FrameCallback): () => void;
    unregisterUpdate(cb: FrameCallback): void;
    start(): void;
    stop(): void;
}
