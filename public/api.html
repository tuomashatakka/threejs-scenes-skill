Clock interface

A simulation time source. Feed it real elapsed seconds via `advance` and run one simulation tick per returned delta. In 'fixed' mode the returned deltas are always exactly `step`, so tick count — not wall time — drives the sim.

interface Clock {
    readonly mode: ClockMode;
    advance(realDelta: number): readonly number[];
    /** Total simulated seconds consumed so far. */
    elapsed(): number;
    reset(): void;
}

AppModule interface

A scene feature in the unidirectional flow. `build` creates objects once; `update` projects the current state onto them every simulation tick. Modules read state — they never write it back.

interface AppModule<S extends object = Record<string, unknown>> {
    name: string;
    build(ctx: SceneContext): void;
    update?(state: S, frame: FrameContext, ctx: SceneContext): void;
    dispose(): void;
}

PropDefinition interface

Declarative description of a reusable prop: how to build its Object3D, plus optional animation clips, embedded lights, and an instancing hint. Author one with `defineProp` and mount it with `createProp` (or via <Prop src>).

interface PropDefinition {
    name: string;
    build(ctx: PropContext): THREE.Object3D;
    clips?(root: THREE.Object3D): THREE.AnimationClip[];
    lights?(root: THREE.Object3D): THREE.Light[];
    instanced?: {
        count: number;
        radius?: number;
        seed?: number;
        place?: InstancePlaceFn;
    };
}
