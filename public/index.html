core

Core scene scaffolding: renderer, frame loop, bootstrap, pointer gestures, disposal, and quality-tier detection. Copy bootstrapScene as the starting point for any new scene.

Exports
ExportSummary
createRendererWebGLRenderer factory: pixel ratio capped at 2, ACES tone mapping, sRGB output, sane shadow defaults.
createFrameLoopSelf-contained Clock-driven loop; subsystems register via onFrame / registerUpdate and share one FrameContext.
bootstrapSceneWires renderer + scene + camera + frame loop + pointer gesture + resize observer + dispose into one call.
attachPointerGestureUnified drag / pinch / wheel / tap layer; identical for mouse, touch and pen.
attachResizeObserverResizes the renderer and camera off the canvas parent (never window.resize).
disposeSceneRecursive teardown that frees GPU memory three.js never auto-disposes.
detectTier / getQualitySettings / QUALITY_PRESETSDetect device tier (mobile / desktop / highEnd) to gate shadows, post-fx, particle budgets and view radius.
 
import { bootstrapScene, disposeScene, detectTier } from 'threejs-scenes'

const app = bootstrapScene({
  canvas,
  onSetup ({ scene, camera, loop }) {
    // build content, register per-frame work via loop.onFrame(...)
  },
})
// later: app.dispose() — or disposeScene(scene)

procedural

Deterministic procedural generation: seeded RNG with per-consumer forking, Poisson-disk scatter, and seamless noise textures.

Exports
ExportSummary
createSeededRngSeeded stream with range / int / pick and fork(label) for deterministic per-consumer sub-streams.
poissonDisk2D Poisson-disk sampling: points with guaranteed minimum spacing, ideal for forests, asteroid fields and star scatter.
createNoiseTextureSeamless tileable value-noise DataTexture sampled on a torus; DOM-guarded (returns null headless).
mulberry32 / hash2 / hash3 / lerp / smoothstepLow-level deterministic primitives backing the RNG and noise helpers.
 
import { createSeededRng, poissonDisk } from 'threejs-scenes'

const rng = createSeededRng(1337)
const trees = rng.fork('trees')           // stable sub-stream
const points = poissonDisk({ width: 100, height: 100, radius: 4, rng: trees })

Exports

post (WebGL)

An EffectComposer factory plus a pass library. The canonical chain is RenderPass → UnrealBloomPass → grade → OutputPass; tone-map exactly once, at the OutputPass.

Exports
ExportSummary
createComposerEffectComposer wired with a DepthTexture so DOF / god-rays / soft particles can sample scene depth.
createGradePass / GradeShaderLinear-HDR colour grade: tint, contrast, saturation, vignette, grain and corner chromatic aberration, before tone-mapping.
createGodRaysPassScreen-space volumetric light shafts radiating from the light's screen position.
createDofPassDepth of field + chromatic aberration in one pass, driven by the circle of confusion (needs the depth texture).
createFilmGrainPassProcedural per-fragment grain (colour or luminance) with optional desaturation.
createRgbShiftPass / createBlockDisplacementPass / createScanCorruptionPassStackable glitch passes for layered datamosh.
createHudBeamTransitionLeft→right luminous beam reveal with a midpoint callback to swap content under the beam.
createStereoRendererAnaglyph (red/cyan) or side-by-side stereo render modes (these bypass the composer).
 
import { createComposer, createGradePass } from 'threejs-scenes'

const composer = createComposer({ renderer, scene, camera })
composer.addPass(createGradePass({ saturation: 1.1, vignette: 0.4 }))
loop.onFrame(() => composer.render())

architecture

Transferable architecture patterns: context-injection modules, pooled materials, lazy procedural textures, undo/redo, never-throw param coercion, and raycast introspection.

Exports
ExportSummary
createSceneModuleWraps a feature as { name, build(ctx), dispose() }; modules receive a SceneContext, never globals.
MaterialPoolCaches materials by key and owns disposal — modules reuse via get(key, factory) and never dispose pooled materials.
EditStackSnapshot-based undo/redo: push before applying edits, restore atomically on failure so no partial state escapes.
createProceduralTexture / proceduralTextureCacheLazy seeded CanvasTexture cache; DOM-guarded to degrade to flat colour headless.
resolveParam / resolveParamsCoerce config- or LLM-driven values: clamp, validate enums, fall back to defaults — never throw.
pickTopLevelRaycast and walk up to the scene's direct child to map a hit back to its owning module.
 
import { createSceneModule, MaterialPool, resolveParams } from 'threejs-scenes'

const pool = new MaterialPool()
const trees = createSceneModule({
  name: 'trees',
  build (ctx) { /* register loop work, add objects */ },
  dispose () { /* free only what this module created */ },
})
