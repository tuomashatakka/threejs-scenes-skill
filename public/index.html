Live demos

Twenty self-contained scenes — pick one from the tab bar to load it full-width below. Feature demos import the library from a local, version-pinned copy (lib/dist/) via importmap; three.js loads from the esm.sh CDN. No build step.

core

Core scene scaffolding: renderer, frame loop, bootstrap, pointer gestures, disposal, and quality-tier detection. Copy bootstrapScene as the starting point for any new scene.

Live demo — minimal-scene.html

Exports
ExportSummary

jsx New in 1.1

A higher-level declarative layer over the library. Author scenes as elements; render() mounts them and drives reactivity from the main frame loop — function-valued props are accessors re-read every frame, so reconciliation is the render tick. Works as JSX or, with no build step, as hyperscript (h). Import from @tuomashatakka/threejs-scenes/jsx.

Live demo — jsx-scene.html

Exports & elements
ExportSummary
renderBootstrap + mount a tree onto a canvas; returns { scene, renderer, loop, getCamera, dispose }.
h / jsx / jsxs / FragmentElement constructors — hyperscript and the transpile-target runtime.
signal / derivedMinimal reactivity; accessors are re-evaluated by the loop.
Intrinsics<scene> <camera> <light> <group> <mesh> <primitive> <prop> <instances> <post>.
